# AI合照生图-智能体

一个温暖的项目，帮助潜在领养者与流浪宠物生成温馨的亲密合照，让领养者提前感受未来与宠物生活的美好憧憬。

## 功能特点

- 📷 上传您的照片
- 🐾 选择流浪宠物照片
- 🎭 选择亲密互动场景
- 🎨 AI生成温馨合照

## 技术架构

```
用户照片 + 宠物照片
       ↓
   GLM-4V 多模态分析
   (提取特征、生成详细提示词)
       ↓
   CogView-3 图像生成
   (生成亲密合照)
       ↓
     最终合照
```

## 安装

1. 安装依赖：

```bash
pip install -r requirements.txt
```

## 使用方法

运行主程序：

```bash
python main.py
```

按提示操作：
1. 选择或输入您的照片
2. 选择或输入宠物照片
3. 选择亲密互动场景
4. 等待生成结果

## 互动场景

| 场景 | 描述 |
|------|------|
| 沙发相拥 | 坐在沙发上，宠物依偎在怀里 |
| 一起睡觉 | 温馨的午睡时光，宠物蜷缩在身边 |
| 喂食时光 | 给宠物喂食的温馨画面 |
| 户外散步 | 阳光下的户外互动 |
| 温柔抚摸 | 轻抚宠物，深情对视 |

## 目录结构

```
AI合照生图-智能体/
├── config.py          # 配置文件
├── main.py            # 主程序入口
├── image_analyzer.py  # 图像分析模块（GLM-4V）
├── image_generator.py # 图像生成模块（CogView）
├── utils.py           # 工具函数
├── requirements.txt   # 依赖
├── pets/              # 宠物照片库
└── output/            # 输出图片
```

## 添加宠物照片

将流浪宠物的照片放入 `pets/` 目录，支持的格式：JPG, JPEG, PNG, GIF, WEBP
